# dev-stream plugins

Official source plugins for [dev-stream](https://github.com/phin-tech/dev-stream).

`registry.json` is the machine-readable catalog. Entries are versioned,
commit-pinned, compatibility-declared, and bound to the SHA-256 of the exact
manifest users review before granting trust.

Install from the dev-stream Settings page using one of these URLs:

- `https://github.com/phin-tech/dev-stream-plugins/tree/main/github`
- `https://github.com/phin-tech/dev-stream-plugins/tree/main/linear`

Plugins run in permission-scoped Deno workers. Review the manifest permissions before trusting a plugin.
